TDS Query API

Description: The TDS Query API provides a RESTful interface for querying Mataray data from a TDS Kaizen using HTTP requests.

Endpoints:

  • GET /data: Retrieves data from the TDS database based on user-specified query parameters.
  • POST /data: Submits a new data point to be added to the TDS database.

Request Parameters:

  • sn (required): Mataray serial number (e.g. "MR-GMP853").
  • param (required): Parameter to query (e.g. "Power").

Response Format:

Responses are returned in JSON format with the following fields:

  • status: The HTTP status code of the response.
  • response: A brief message describing the result of the request.
